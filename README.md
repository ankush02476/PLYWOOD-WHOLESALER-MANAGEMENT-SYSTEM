# 🪵 Plywood Wholesaler Management System

## 📌 Overview

The **Plywood Wholesaler Management System** is a desktop-based application developed using Java, SQL, Java Swing, and AWT.
It is designed to manage and automate the daily operations of a plywood wholesale business, including inventory, sales, employees, and accounts.

---

## 🔐 User Authentication System

The application includes a secure login and registration system:

* When the application starts, a **Login Page** appears
* Existing users can enter their **username and password** to access the system
* New users can click on the **Register button** to create an account
* After registration, users can log in using their credentials

---

## 🖥️ Application Flow

1. User opens the application
2. Login page is displayed
3. User logs in or registers
4. After successful login, the **Main Dashboard** opens
5. A **left-side panel** displays all module names with icons
6. Clicking on any module opens its respective page

---

## 🚀 Key Features

* 🔐 Login & Registration system
* 📦 Product and stock management
* 🧾 Purchase and sales tracking
* 👥 Employee management
* 🏢 Supplier management
* 💰 Account handling
* 📊 Report generation
* 🖨️ Print functionality available in all modules

---

## 🧩 Modules Included

* **Product Module** – Manage product details
* **Purchase Module** – Record purchases
* **Sell Module** – Manage sales and billing
* **Stock Module** – Track inventory
* **Employee Module** – Maintain employee records
* **Account Module** – Manage financial transactions
* **Supplier Module** – Store supplier information
* **Report Module** – Generate reports

👉 Each module includes a **Print button** to print data easily.
